Software

TT-Forge

End-to-end MLIR compiler. Compile models from PyTorch, JAX, and ONNX — the recommended entry point.

TT-NN

High-level Python API of pre-optimized neural-network operations.

TT-Metalium™

Low-level C++ SDK for writing custom kernels on Tensix cores.

TT-Lang

Python DSL for authoring fused custom operations.

TT-MLIR

Core MLIR compiler defining the TTIR, TTNN, and TTKernel dialects.

Tools

TT-Inference-Server

Deploy and serve models — the canonical source for validated model support.

TT-Studio

Point-and-click web UI for deploying a model — the fastest path for new users.

vLLM

Tenstorrent's high-throughput LLM serving engine fork.

TT-SMI

Command-line device telemetry and firmware info — ships with the software stack.

TT-Toplike

Real-time terminal monitor: power, temperature, current, and core activity.

TT-NN Visualizer

Inspect model execution: graphs, memory plots, tensor and buffer views.

Model-Explorer

Hierarchical, expandable view of model graphs.

TT-Topology

Configure Ethernet routing on multi-card QuietBox and LoudBox systems.

Guided VS Code learning environment — browse tracks in the Learn section above.
