Software

TT-Forge™

Tenstorrent's end-to-end MLIR compiler. Compile models from PyTorch, JAX, and ONNX — the recommended entry point for running existing models.

TT-NN™

High-level Python API of pre-optimized neural-network ops.

TT-Metalium™

Low-level C++ SDK for writing custom kernels on Tensix cores.

TT-Lang™

Python DSL for authoring fused custom operations.

TT-MLIR

Core MLIR compiler defining the TTIR, TTNN, and TTKernel dialects.
