ttnn.rsqrt

ttnn.rsqrt(input_tensor: ttnn.Tensor, *, fast_and_approximate_mode: bool = False, memory_config: ttnn.MemoryConfig | None = None) → ttnn.Tensor

Applies rsqrt to input_tensor element-wise.

\[\begin{split}rsqrt(\\mathrm{input\\_tensor}_i)\end{split}\]
Args:

  • input_tensor

Keyword Args:

  • fast_and_approximate_mode (bool): “Use fast and approximate mode”.

  • memory_config (Optional[ttnn.MemoryConfig]): Memory configuration for the operation.

Example:

>>> tensor = ttnn.from_torch(torch.tensor((1, 2), dtype=torch.bfloat16), device=device)
>>> output = ttnn.rsqrt(tensor, fast_and_approximate_mode=true)
