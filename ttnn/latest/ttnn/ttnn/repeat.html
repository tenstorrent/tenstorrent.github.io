ttnn.repeat

ttnn.repeat(input_tensor: ttnn.Tensor, shape: ttnn.Shape) → ttnn.Tensor

Returns a new tensor filled with repetition of input input_tensor according to number of times specified in shape.

Args:

  • input_tensor: the input_tensor to apply the repeate operation.

  • shape: The number of repetitions for each element.

Keyword Args:

  • memory_config: the memory configuration to use for the operation

Example:

>>> tensor = ttnn.repeat(ttnn.from_torch(torch.tensor([[1, 2], [3, 4]]), 2,)), device)
>>> print(tensor)
tensor([[1, 2],
[1, 2],
[3, 4],
[3, 4]])
