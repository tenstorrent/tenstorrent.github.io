ttnn.to_layoutÔÉÅ

ttnn.to_layout(tensor: ttnn.Tensor, layout: Layout, dtype: DataType | None = None, memory_config: MemoryConfig | None = None) ‚Üí ttnn.TensorÔÉÅ

Organizes the ttnn.Tensor tensor into either ROW_MAJOR_LAYOUT or TILE_LAYOUT. When requesting ROW_MAJOR_LAYOUT the tensor will be returned unpadded in the last two dimensions. When requesting TILE_LAYOUT the tensor will be automatically padded where the width and height become multiples of 32. In the case where the layout is the same, the operation simply pad or unpad the last two dimensions depending on layout requested.

Args:

  • tensor: the ttnn.Tensor

  • layout: the layout of either ttnn.ROW_MAJOR_LAYOUT or ttnn.TILE_LAYOUT.

  • dtype: the optional output data type.

  • memory_config: the optional output memory configuration.

  • device: Device/DeviceMesh whose worker thread on host should be used for the layout conversion

Example:
>>> device_id = 0
>>> device = ttnn.open_device(device_id=device_id)
>>> tensor = ttnn.to_device(ttnn.from_torch(torch.randn((10, 64, 32), dtype=torch.bfloat16)), device)
>>> tensor = ttnn.to_layout(tensor, layout=ttnn.TILE_LAYOUT)
>>> print(tensor[0,0,:3])
Tensor([ 1.42188, -1.25, -0.398438], dtype=bfloat16 )
