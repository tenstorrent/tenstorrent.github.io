Tenstorrent docs — offline bundle (includes TT-LoudBox Blackhole)
================================================================

STEP 1 — Extract this zip to a folder (e.g. Desktop or Documents).
        Do not open files from inside the zip.

STEP 2 — Open index.html in your browser (double-click it or drag it into
        the browser window). Styles and formatting are built in, so no
        web server is required.

The TT-LoudBox (Blackhole) section is under: Systems → TT-LoudBox (Blackhole).

If anything does not look right (e.g. missing styles), try opening index.html
from the extracted folder rather than a subpage, or run a local server:
  python3 -m http.server 8000   (then open http://localhost:8000)
